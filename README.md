# Tableau-Retail-Sales-Analysis

### 📊 Sales & Profit Performance Dashboard – Tableau Project
#### 🧠 Project Overview
This project presents an interactive Sales & Profit Performance Dashboard built in Tableau. It provides a deep dive into sales data across multiple dimensions — including category, product, region, time, and customer demographics. The goal was to uncover actionable insights that can help drive business decisions.

The dashboard is split into two main views:

- Deep Analysis Dashboard – Focuses on product performance, weekly trends, and category-level quantity distribution.

- Sales Summary Dashboard – Highlights high-level KPIs, regional performance, gender-based ordering patterns, and profit trends over time.

### 📁 Datasets Used
Sales transactions (orders, quantity, revenue, profit)

Product catalog (categories, product IDs)

Customer demographics (age groups)

Geographic data (region, state, city)

Time-series data (monthly & weekly) 

### 🔍 Key Insights
#### 📈 Deep Analysis Dashboard
Top Category by Quantity: Electronics leads with 10,419 units, followed by Hardware (8,801) and Home Appliances (6,782).

Top 5 Products: Widget AL, Device ID, Tool BA, Accessory T, Accessory AN.

Weekly Profit Margins: Profit % remains stable between 13.82% – 15.13%, with Wednesday showing the highest profit % (15.13%) despite slightly lower order volume.

Monthly Orders Trend: Orders and quantity follow a similar seasonal pattern, peaking mid-year and tapering toward December.

Customer Age Group Contribution:

51–70 yrs → 3,005 orders

31–50 yrs → 1,854 orders

18–30 yrs → 1,276 orders

Navigation link between dashboards (To Sales Summary Dashboard)


### 🚀 Business Recommendations
Focus marketing efforts on Electronics & Hardware – highest quantity categories.

Investigate Wednesday’s high profit % – replicate across other weekdays.

Target older age groups (51–70) – highest order volume.

Review low-performing products (Widget DU → Widget BA) – consider bundling or discounting.

Expand in East Coast & South – high order volume, but monitor profit margins.
